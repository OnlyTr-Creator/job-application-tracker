## ðŸ”§ Troubleshooting

### "Can't find application to update"
**Cause**: Company Name or Job Title doesn't match exactly

**Solution:**
1. Export data to see exact names
2. Copy-paste from CSV
3. Watch for: extra spaces, caps, punctuation

### "Success Score seems wrong"
**Cause**: Score factors in time passed

**Solution:**
- Recent applications: Score is accurate
- Old applications: Score decreases over time (this is intentional!)
- To see original score: Update status to re-calculate

### "Calendar events not creating"
**Cause**: Google Calendar not authorized

**Solution:**
1. CREAO will prompt for authorization
2. Follow OAuth flow
3. Grant calendar access
4. Try again

### "Dashboard shows 0 applications"
**Cause**: CSV file empty or corrupted

**Solution:**
1. Export data
2. Check job_applications.csv exists
3. Make sure you've added at least one application

### "AI suggestions not helpful"
**Cause**: Not enough data yet

**Solution:**
- Add at least 5 applications for meaningful insights
- Update statuses regularly
- Track for at least 2 weeks before analyzing patterns

## â“ Common Questions

**Q: How many applications should I track?**
A: Track EVERY application. Even "unlikely" ones. Better data = better decisions.

**Q: Should I mark old applications as "Rejected"?**
A: Yes. If >30 days with no response, likely rejected. Move on and focus energy elsewhere.

**Q: Can I track applications from before I started using this?**
A: Yes! Just backfill with approximate dates. Better to have historical data.

**Q: How often should I follow up?**
A:
- After applying: 7-10 days
- After interview: 5-7 days (unless they gave timeline)
- After offer: Within their deadline (usually 3-7 days)

**Q: What's a good interview rate?**
A:
- **Excellent**: >25%
- **Good**: 15-25%
- **Average**: 10-15%
- **Needs work**: <10%

**Q: Should I apply to jobs I'm not fully qualified for?**
A: If you meet 70% of requirements, yes! Track the outcome to learn what works.

**Q: How do I handle recruiter spam?**
A: Only track roles you're genuinely interested in. Quality > Quantity.
